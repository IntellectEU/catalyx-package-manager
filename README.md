#  Catalyx Package Manager

## Get Started

Catalyx Package Manager (CPM) is an application registry designed to enable users of the Canton Network to discover, distribute, and download applications.

CPM provides a streamlined, user-friendly console where organizations can:

Discover - Explore a list of applications shared by app builders on the Canton Network

Distribute - Publish your own applications, request audits, and control who has access

Download - Download and run powerful Canton Network applications

Get started today and discover how you can use Catalyx Package Manager to discover, distribute, and download applications on the Canton Network.

To get started with CPM please check the CPM [User guide](https://docs.catalyst.intellecteu.com/cpm/CPM/onboarding-guide).
